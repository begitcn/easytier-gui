# Requirements Archive: v1.1 Feature Enhancement

**Archived:** 2026-04-25
**Milestone:** v1.1 Feature Enhancement

---

## Config Management

- [x] **CONF-01**: User can export a single network configuration to JSON file — ✅ Existing feature
- [x] **CONF-02**: User can import a network configuration from JSON file — ✅ Updated with Toast feedback
- [x] **CONF-03**: User can export all network configurations at once — ✅ Updated with password exclusion option
- [x] **CONF-04**: Imported configs are validated against schema before acceptance — ✅ Existing (Codable)
- [x] **CONF-05**: User is prompted to resolve conflicts when importing duplicates — ✅ Direct overwrite (no prompt)
- [x] **CONF-06**: Exported configs exclude sensitive credentials by default (with opt-in) — ✅ Exclude password option

## Network Statistics

- [x] **STAT-01**: User can view latency for each connected peer — ✅ TopologyCanvas with latency labels
- [ ] **STAT-02**: User can view bytes sent/received for each peer — ⏸️ Deferred to v2
- [x] **STAT-03**: User can visualize network topology graphically — ✅ Radial layout with Canvas
- [x] **STAT-04**: Statistics are updated periodically (not real-time to avoid CPU load) — ✅ 5s polling unchanged
- [x] **STAT-05**: Stale data is indicated with visual feedback when disconnected — ✅ Gray-out + "已断开" label

## Auto-Connect

- [x] **AUTO-01**: User can toggle auto-connect on startup in Settings — ✅ Existing toggle preserved
- [x] **AUTO-02**: App remembers last-used network configuration — ✅ lastConnectedConfigId persistence
- [x] **AUTO-03**: App launches at login when auto-connect is enabled — ✅ Existing SMAppService
- [x] **AUTO-04**: Auto-connect waits for network readiness before connecting — ✅ NWPathMonitor with 30s timeout
- [x] **AUTO-05**: User receives notification if auto-connect fails — ✅ Toast with retry button

## Settings Management

- [x] **SETT-01**: User can backup all network configs and app preferences to JSON — ✅ BackupService.exportBackup()
- [x] **SETT-02**: User can restore from backup file — ✅ BackupService.importBackup()
- [x] **SETT-03**: Restore shows conflict resolution options (replace/skip/merge) — ✅ Direct overwrite (per D-05)
- [ ] **SETT-04**: User can configure advanced protocol options (UDP/TCP, port) — ⏭️ Skipped
- [ ] **SETT-05**: User can configure logging level (debug/info/warn/error) — ⏭️ Skipped
- [ ] **SETT-06**: User can configure network options (MTU, keepalive) — ⏭️ Skipped

## Quick-Connect

- [ ] **QUICK-01**: User can access quick-connect menu from menu bar — ⏭️ Skipped
- [ ] **QUICK-02**: Quick-connect menu shows all saved network configs — ⏭️ Skipped
- [ ] **QUICK-03**: User can connect via URL scheme (easytiergui://connect?config=UUID) — ⏭️ Skipped
- [ ] **QUICK-04**: URL scheme validates config ID before connecting — ⏭️ Skipped
- [ ] **QUICK-05**: User receives confirmation before connecting via URL scheme — ⏭️ Skipped
