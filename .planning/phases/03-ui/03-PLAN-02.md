## Task 1: 添加键盘快捷键处理

**Type:** execute

<read_first>

- EasyTierGUI/Services/ProcessViewModel.swift (AppTab 枚举定义，第562-584行)

<action>
在 ContentView 中添加键盘快捷键支持。使用 SwiftUI 的 `.focusable()` 修饰符和 `onKeyPress` 来捕获键盘事件：

1. 在 `ContentView` struct 中添加 `@FocusState` 变量用于焦点管理：
```swift
@FocusState private var isFocused: Bool
```

2. 在 NavigationSplitView 上添加 focusable 修饰符：
```swift
.navigationSplitView {
    SidebarView(selectedTab: $selectedTab)
        .navigationSplitViewColumnWidth(min: 180, ideal: 200, max: 260)
        .focused($isFocused)
} detail: {
    DetailView(selectedTab: selectedTab)
}
.onAppear { isFocused = true }
.onKeyPress(.escape) {
    isFocused = true
    return .handled
}
```

3. 添加快捷键处理：
```swift
.onKeyPress(keys: [KeyPressEquivalent.one]) {
    selectedTab = .connection
    return .handled
}
.onKeyPress(keys: [KeyPressEquivalent.two]) {
    selectedTab = .peers
    return .handled
}
.onKeyPress(keys: [KeyPressEquivalent.three]) {
    selectedTab = .logs
    return .handled
}
.onKeyPress(keys: [KeyPressEquivalent.four]) {
    selectedTab = .settings
    return .handled
}
.onKeyPress(keys: [KeyPressEquivalent.r]) {
    Task { @MainActor in
        await vm.refreshPeers()
    }
    return .handled
}
```

注意：由于 SwiftUI 的 onKeyPress API 限制，使用 FocusState + 按键监听方式实现。
</action>
