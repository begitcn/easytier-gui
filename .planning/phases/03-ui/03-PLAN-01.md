# Plan 03-01: 统一间距规范

<objective>
按照 macOS HIG 8pt grid 系统统一所有视图的间距，使界面视觉更协调一致
</objective>

<tasks>

## Task 1: 定义间距常量

**Type:** execute
**Files:** EasyTierGUI/Views/ContentView.swift

<read_first>
- EasyTierGUI/Views/ContentView.swift (当前间距使用情况)
- EasyTierGUI/Views/ConnectionView.swift (卡片间距)
- EasyTierGUI/Views/PeersView.swift (工具栏间距)
- EasyTierGUI/Views/LogView.swift (日志列表间距)
</read_first>

<action>
在 ContentView.swift 文件末尾添加间距常量扩展：

```swift
// MARK: - Spacing Constants (macOS HIG 8pt Grid)

extension CGFloat {
    /// 组件内间距: 8pt
    static let spacingXS: CGFloat = 8
    /// 组件内间距: 12pt
    static let spacingS: CGFloat = 12
    /// 组件内间距: 16pt
    static let spacingM: CGFloat = 16
    /// 组件外间距: 20pt
    static let spacingL: CGFloat = 20
    /// 组件外间距: 24pt
    static let spacingXL: CGFloat = 24
    /// 卡片内边距: 20pt
    static let cardPadding: CGFloat = 20
    /// 卡片间距: 24pt
    static let cardSpacing: CGFloat = 24
}
```

然后更新 ContentView.swift 中的 SidebarView 间距：
- `.padding(.vertical, 8)` → `.padding(.vertical, .spacingXS)`
</action>

<verify>
- [ ] grep -n "spacingXS\|spacingS\|spacingM\|cardPadding" EasyTierGUI/Views/ContentView.swift
</verify>

<acceptance_criteria>
- `EasyTierGUI/Views/ContentView.swift` 包含 `static let spacingXS: CGFloat = 8`
- `./build.sh` 编译成功
</acceptance_criteria>

</tasks>
